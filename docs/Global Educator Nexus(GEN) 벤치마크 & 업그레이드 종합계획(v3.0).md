**Global Educator Nexus(GEN) 벤치마크 & 업그레이드 종합계획(v3.0)**

------

## 1. 비전 & 목표 상태(Enterprise-grade GEN)

두 문서가 합의하는 목표 상태는 한 줄로 이렇습니다.

> **“교육자(ESL+국제학교) 채용 전 과정을 AI가 지원·자동화하고, 대규모/다국적/규제환경에서도 안정적으로 운영되는 ‘교육자 전용 Talent Intelligence + ATS-lite + Agency Marketplace’ 플랫폼”** 

이를 구성하는 5대 전제(=엔터프라이즈 표준)는 다음과 같습니다.

1. **Agentic AI + Skills-based Hiring**
    단순 추천이 아니라 소싱→스크리닝→면접→오퍼→온보딩까지 **AI가 파이프라인을 능동 최적화**하는 구조. 
2. **멀티모달 AI + 편향/윤리/프라이버시 내장**
    텍스트 임베딩(이력서/JD) + 비디오/음성 신호를 **하나의 매칭 엔진에서 결합**, 동시에 편향·AI 개인정보 리스크를 관리. 
3. **클라우드-네이티브 확장성 + 통합 생태계**
    현대 엔터프라이즈 잡보드(LinkedIn/Indeed/SmartRecruiters/Greenhouse/Lever)는 **ATS/CRM/커뮤니케이션/평가툴 통합 + 자동 확장(오토스케일, 큐, CDN)**을 기본으로 둠.  
4. **Consumer-grade CX/RX + 모바일 퍼스트**
    구직자는 “원클릭 지원/상태 타임라인/모바일 최적”, 채용자는 “협업/알림/대시보드”가 표준 UX.  
5. **글로벌+한국 규제 동시충족(Compliance-first)**
    PIPA/GDPR/채용절차법/향후 ISMS-P까지 **기능이 아니라 아키텍처에 내장**.

## 2. 벤치마크 인사이트 → GEN에 적용할 “핵심 원리”

문서②(글로벌 동향)에서 추출되는 **엔터프라이즈 잡보드 공통 성공원리**를 GEN 식으로 변환하면:

| 글로벌 엔터프라이즈 동향        | 의미                         | GEN 적용 원리                                         |
| ------------------------------- | ---------------------------- | ----------------------------------------------------- |
| AI 매칭/추천이 “코어”           | 추천 정확도=플랫폼 생존      | **멀티모달/에이전트형 AI를 GEN 핵심 차별화로 고도화** |
| ATS+CRM 통합                    | 잡보드 단독으론 LTV 낮음     | **ATS-lite+Agency CRM 기능을 기본 번들**              |
| Talent Insights/대시보드        | 데이터 기반 채용이 표준      | **학교/에이전시용 KPI·퍼널·예측 대시보드 구축**       |
| Conversational UI/모바일 퍼스트 | 지원 마찰 제거가 전환율 결정 | **원클릭 지원·PWA·푸시·챗봇 내장**                    |
| 글로벌/다국어 + 규제            | 국가 확장=컴플라이언스 확장  | **레인(ESL/Intl) 분리형 i18n·정책·검증 체계**         |

## 3. GEN 현행 포지션과 핵심 갭

### 3-1. 강점(지켜야 할 코어)

- **교육자 도메인 특화(ESL+국제학교)**
- **AI 임베딩 매칭 + 비디오 이력서 분석 + 비자 체크**
- Next.js/Prisma/pgvector 등 현대적 스택 기반  

### 3-2. 엔터프라이즈 대비 갭(이번 업그레이드의 이유)

- **스케일/성능**: 오토스케일, 파티셔닝, 비디오 CDN, 대량 알림 큐 부족
- **통합/생태계**: ATS/CRM/Zoom-Meet/Slack/Zapier 웹훅 등 부족
- **고급 UX/RX**: 실시간 협업, 모바일 원클릭, 채팅/챗봇/알림 고도화 필요
- **보안/프라이버시**: MFA/감사로그/AI 데이터 익명화/삭제정책 미흡
- **인사이트/대시보드**: 학교·에이전시용 KPI/퍼널/예측 분석 부족

## 4. 업그레이드 “9대 Pillar” 통합 백로그

(= 글로벌 동향을 GEN 제품/아키텍처로 바꾼 실행 단위)

### P1) AI/Matching → **Talent Intelligence(Agentic AI)**

- **멀티모달 매칭**: 텍스트 임베딩 + 비디오/음성 벡터 통합
- **편향 감지/DEI-safe scoring**
- 에이전트 3종: **JD 생성/개선**, **소싱/리디스커버리**, **스케줄링/커뮤니케이션 자동화** 

### P2) Discovery 고도화

- **하이브리드 검색**(BM25 + kNN + RRF)
- pgvector 유지 + **Elasticsearch 벡터 인덱스 병행**(대용량/복합필터 대응) 

### P3) ATS-lite + 협업 파이프라인

- 칸반 파이프라인(NEW→SCREENING→INTERVIEW→OFFER→HIRED)
- **실시간 협업(코멘트/@멘션/평가표)**
- 국제학교/교육청 조직을 고려한 **다단계 승인 워크플로우 빌더** 

### P4) 구직자 CX 혁신

- **원클릭 지원 + 자동 파싱(이력서/LinkedIn 등)**
- **PWA/푸시/오프라인/다크모드**
- 지원 타임라인·자동 피드백·개인화 추천 홈 

### P5) 채용자 RX & 인사이트

- 실시간 KPI(지원률/전환/Time-to-Hire/오퍼수락률/Source-of-Hire)
- 학교/에이전시 단위 벤치마크 + CSV/BI Export
- **예측 분석(수요/충원 기간)**  

### P6) Integrations / Marketplace

- **웹훅·Public API Surface 확대**(Server Actions 편중 완화)
- Zoom/Meet/Teams 자동 링크, Slack/Teams/KakaoWork 알림봇
- Zapier/Make 커넥터로 노코드 연동 확장 

### P7) 아키텍처/스케일 전환

- 도메인 분리형 MSA(Job/Candidate/Application/Assessment/Notification)
- **이벤트 기반 + Saga 패턴**
- K8s(또는 Vercel+AWS 하이브리드), 비디오 CDN, 캐시 계층 확장 

### P8) 보안/컴플라이언스

- **MFA/SSO, 감사로그, 임베딩·비디오 데이터 익명화 정책**
- 채용서류 자동 파기/보관 스케줄러(PIPA/GDPR/채용절차법)
- ISMS-P/SOC2 로드맵 포함 

### P9) 교육자 특화 혁신

- **자격/라이선스 검증 배지(TEFL/PGCE/IB 등)**
- (선택) VR/데모수업 평가
- 국가별 비자 자동화 체크리스트+API

## 5. 통합 로드맵(6–12개월)

> 두 문서의 3-Phase(빠른 고도화) + 4-Phase(엔터프라이즈 전환)를 하나로 겹쳐 설계. 

### Phase 0 / 0–4주: **안정화 & 기술부채 청산(v1.1)**

- Seed/더미/중복 가입 로직 정리
- 누락 API 완성, OAuth 재활성화
- 테스트 커버리지 80%
   **산출물**: 온보딩 완성, 99% uptime 기반

### Phase 1 / 1–2개월: **기반 강화 & 전환율 개선**

- PWA/모바일 퍼스트/다크모드
- 원클릭 지원(파싱→프로필 자동화)
- 기본 실시간 통계 대시보드
- MFA 1차(관리자/학교)

### Phase 2 / 3–7개월: **코어 엔터프라이즈 기능 + AI Boost(v2.0)**

- 멀티모달 AI 매칭 + 편향감지
- ATS-lite 칸반/협업/평가표
- 실시간 채팅/알림(SMS 포함)
- 20+ 외부 연동(ATS/CRM/영상면접/웹훅)
- ES 하이브리드 검색 엔진

### Phase 3 / 8–10개월: **아키텍처 전환 & 인사이트 고도화(v3.0 ready)**

- MSA 단계 분해 + 이벤트/Saga
- 대규모 알림 큐/워커/DLQ
- School/Agency 인사이트(퍼널/소스/예측) 완성
- 부하/통합/보안 테스트

### Phase 4 / 11–12개월(옵션): **데이터/컴플라이언스/혁신**

- 데이터 정제·중복 병합·규제비준수 제거
- 이력서 재파싱 → 구조화 자산화
- 블록체인 배지/VR/게임화 등 고급 기능
- ISMS-P/SOC2 준비 착수
